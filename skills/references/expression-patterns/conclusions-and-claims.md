# Conclusions and Claim Calibration

 module is designed to be read on its own.

## Recommended Expressions

| Scenario | Expression | 中文说明 |
|----------|------------|----------|
| Main conclusion | `Taken together, these findings suggest that [conclusion].` | 适合结论开头，语气稳健。 |
| Contribution statement | `This study contributes to [field] by [specific contribution].` | 直接说贡献内容，不要空泛自夸。 |
| Practical implication | `These findings have practical implications for [audience / task].` | 适合 discussion 或 conclusion 中连接实践价值。 |
| Scope boundary | `The conclusions should be interpreted within the context of [boundary].` | 给结论加边界。 |
| Future work | `Future work could extend this analysis by [next step].` | 写 future work 时避免过度承诺。 |

## Avoid Expressions

| Avoid | Better Direction | 中文说明 |
|-------|------------------|----------|
| `This paper completely solves ...` | State what the study improves or clarifies | 避免过度结论化。 |
| `This is the first study ever to ...` | Only use if verifiable and necessary | “首次”很容易失真。 |
| `Groundbreaking / revolutionary` | Describe the concrete value | 避免夸张形容词。 |
| `There is no doubt that ...` | Use calibrated claim language | 结论中需要保留学术谨慎。 |

## Usage Scenarios

- 结论首段总结主要发现。
- contribution 段说明方法、数据或理论上的具体增量。
- discussion 尾段连接政策、规划或方法论启示。
- future work 段写明确可扩展方向，而不是泛泛而谈。
