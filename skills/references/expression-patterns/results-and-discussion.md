## Recommended Expressions

| Scenario | Expression | 中文说明 |
|----------|------------|----------|
| Main result | `The results show that [finding].` | 最基础也最稳妥的结果句式。 |
| Quantified comparison | `[Method] outperforms [baseline] by [value] on [metric].` | 明确比较对象、指标和提升幅度。 |
| Pattern interpretation | `This pattern suggests that [interpretation], particularly in [context].` | 讨论段用 `suggests` 控制语气。 |
| Heterogeneity | `The effect is more pronounced in [group / area / condition].` | 适合说明异质性或分组差异。 |
| Limitation-aware discussion | `This result should be interpreted in light of [constraint].` | 讨论时保留边界意识。 |

## Avoid Expressions

| Avoid | Better Direction | 中文说明 |
|-------|------------------|----------|
| `The results are very good.` | State the actual metric or comparison | “好”没有学术信息量。 |
| `This proves that ...` | Use `shows`, `suggests`, or `indicates` | 学术论文里通常避免过强证明语气。 |
| `Obviously, ...` | Remove the judgment word | 避免把解释写成理所当然。 |
| `It is worth noting that ...` | Note the point directly | 过度使用会显得冗余。 |

## Usage Scenarios

- 结果段第一句概括核心发现。
- 表格和图像后的解释句，用于把数值结果转成学术含义。
- 讨论段比较不同城市、样本组或实验条件。
- 讨论局限时，避免把结果说得超出证据范围。
